# 결과 정리 (2026-07-20) · 알래스카 내부 증강·얕은 3D·timelapse

오늘 사이클(Phase 1 재검증 → W2.1/KPDC → W3 → 알래스카 내부 실험) 종합. 모든 수치는 공간블록 CV·실측 held-out 기준. 회의적 검증 반영.

## 1. 알래스카 내부 라벨 증강 × 다중 DL · **적대적 검증 후 대폭 정정**

스크립트 `aug_within_alaska.py`, 결과 `aug_within_alaska_{results,sweep,oof}.csv`, 그림 `outputs/figures/12_aug_alaska/`.
1차 보고(GBM 14.96→14.20, MLP 13.98→13.86 등 "4모델 전부 유의 개선")는 **적대적 검증(major_concerns)에서 대부분 기각**됐다.

**검증에서 드러난 문제**:
1. **GBM 개선은 누설이었다**: pseudo 라벨 공변량을 test 블록 앵커 셀에서 복사(최근접)해 test 경계 바로 밖(~2.4km)에 배치. donor가 test 블록인 pseudo 제거 시 14.22→**15.99**로 real_only(14.96)보다 악화. 개선 전량이 test 인접 특징복제 채널 의존.
2. **MLP "유의"는 불성립**: 셀 단위 iid 부트스트랩이 공간 자기상관 무시. 74블록 블록 부트스트랩 CI [−0.53, +0.32]. 또 13.86은 전역 RNG 운(fresh seed=0에서 14.45로 real_only보다 나쁨). seed 평균 이득 ≈0.09cm(잡음 0.4-0.5 미만).
3. **표본 과대 표기**: "13,606셀"은 74블록·고유 1km 위치 426개·단일 블록 35%. "13.86 SOTA 대역 하단" 주장 철회(σ_y=17.4cm 기준 skill 0.20 수준, 문헌과 지지척도 상이).

**검증 후 살아남은 결론**:
- **MLP(real_only) > GBM은 3-seed 전부 유지**(≈−0.7cm). 알래스카 내부에서 DL이 GBM 소폭 상회는 성립.
- **Stefan 유사라벨은 placebo 대비 실제 정보 보유**(평균/셔플 라벨로 바꾸면 크게 악화). 물리 prior 유효성 자체는 재확인.
- TabM에서 증강이 seed 분산을 줄임(안정화 효과). 정확도 개선 주장은 불가.
- **종합: "증강이 알래스카 내부 정확도를 개선한다"는 현재 근거로 성립하지 않는다.** 성립 조건(거리 버퍼·블록 부트스트랩·multi-seed·nested 선택)으로 재실행해야 판정 가능.

## 2. 얕은 3D 지중온도장 (알래스카 0-3m, 검증 완료)

스크립트 `shallow3d_alaska.py`, 그림 `outputs/figures/14_shallow3d/`.
- 라벨: 알래스카 시추공 0-3m 764행·14사이트(심부판의 스위스 편중 해소, 전부 알래스카).
- 필드 검증: 연최대 지중온도 RMSE 2.66°C, R² 0.47, skill 0.27. 단조제약 위반율 0.000.
- LORO 8/10 사이트 양의 skill.
- **0°C→ALT 정합: r 0.28, RMSE 41cm** (심부판 r 0.16·172cm 대비 개선). 단 R² −0.29·bias +11cm로 절대 정합은 미완(얕은 사이트 과대·깊은 사이트 과소).

**결론**: 얕은 3D는 0°C→ALT 상관이 심부판보다 높고(r 0.28 vs 0.16) 알래스카 실측 기반이나, ALT 유도 라벨로 쓰기엔 아직 노이즈가 크다(증강 소스로는 신중).

## 3. timelapse ALT (완료, `timelapse_alaska_results.csv`)

2010-2024 연도별 알래스카 ALT(99,931 관측·연도별 forcing). 그림 `outputs/figures/13_timelapse/`, 애니메이션 `outputs/animations/timelapse_alt_alaska.gif`(프레임 15장).

| 평가 | STEFAN(물리) | GBM_ANN | MLP |
|---|---|---|---|
| 연도 홀드아웃(leave-one-year-out) | **14.97 (R² 0.34, skill +0.19)** | 19.19 | 16.27 |
| within-site 연도 anomaly(시간 신호) | corr **+0.06**, R² −0.01 | corr −0.09 | corr +0.05 |

- **연도별 지도 산출은 가능**하고 물리 forcing이 최선(연도 홀드아웃 14.97cm). T-lite 결론과 정합.
- **정직한 한계**: 시간 분산은 전체의 40%(anomaly SD 11.7cm)나, **연도 간 변동 자체는 어떤 모델도 예측 못 한다**(corr ≈0.06). corr(ΔTDD, ΔALT)=+0.07로 그해 더위-그해 융해 연결이 관측 수준에서 매우 약함(측정시점·미세프로세스 잡음 추정).
- 추세: 2010-2024 +1.29cm/yr(R² 0.07, 약함·표본구성 변동 주의).
- **프레이밍**: "연별 ALT 예측"이 아니라 "물리 forcing 연별 지도 + 시간 신호의 예측 불가능성 정량화"가 정직한 산출.

## 4. 오늘 확정된 큰 그림 (검증 반영)

| 축 | 결론 |
|---|---|
| 전이(OOD) | 모델 구조로 못 뚫음. 물리 상수 E가 앵커. 모든 모델 레나 skill 음수 |
| 내삽(알래스카) | MLP>GBM 소폭(−0.7cm)은 성립. **증강 개선은 누설·seed 착시로 기각(재실험 필요)** |
| 시간축 | 연별 지도는 물리 forcing으로 가능(14.97cm). 연도 간 anomaly는 예측 불가(corr 0.06) |
| 공변량 추가 | 내삽 이득·전이 손실(covariate shift) |
| 3D | 심부=스위스 외삽(기각), 얕은=알래스카 실측(개선되나 ALT 유도 미완 r 0.28) |
| 물리 결합 | 잔차·파라미터예측·미분물리층 전부 전이서 무익. 물리 상수만 유효 |
