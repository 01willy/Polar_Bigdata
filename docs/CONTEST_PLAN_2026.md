## 3. 핵심 주장 3개와 실험 위계 (GPT v2 채택)

모든 실험은 아래 위계 중 하나에 배정되고, 어디에도 배정되지 않으면 하지 않는다.

- **Primary claim**: ALT 2D + shallow 3D 열구조 + AOA + 보정 UQ를 관측기반으로 산출한다.
- **Mechanism claim(정보병목)**: 무엇이 ALT 예측을 지배하는가(기후·지형·SAR·토양·식생·위치성). 병목은 모델 용량이 아니라 공변량 정보량이다.
- **Reliability claim(신뢰가능성)**: 어디서 믿을 수 있고 어디서 외삽인가(누설통제 평가·AOA·보정 커버리지).
- **Extension claim**: T-lite 예측, 4D, foundation embedding, 3D/4D operator learning(전부 조건부·게이트).

실험 기록 규칙: 새 실험마다 "지지 주장 / 입력 / 타깃 / baseline / 성공조건 / 중단조건 / 출력 CSV·그림"을 로그에 남긴다. 결과는 성공조건 통과 여부와 함께 본문·부록·future work로 분류한다.

## 5. 비가역 하한과 얕은 ALT 상대오차 (질문 9 답)

### "SOTA 12cm 아래는 무의미한가" — 점검증 비가역 하한은 실재한다

약 10~12cm의 비가역 하한이 존재하며, 이는 모델 한계가 아니라 검증 프로토콜(점 vs 면적평균)과 관측 정밀도가 만든다.

- 최고 성능 P-band 업스케일도 화소 hold-out 7.5~9.1cm 대비 독립 현장 in-situ는 11~12cm. 이 3~4cm 격차는 점관측이 화소 면적평균을 대표 못 하는 대표성 오차.
- 물리 하한 근사: 단일 프로브 ±3cm와 셀 내 자연 공간변동(툰드라 std 9~12cm)을 RSS 결합하면 약 9.5~12.4cm. 완벽한 모델도 단일점 평가에서 이 노이즈를 못 없앤다. ABoVE가 화소당 20~30 프로브 평균을 요구하는 이유.
- 우리 위치: 15.72~16.95cm는 이 하한에 근접. 모델 정교화로 12cm 아래는 원리적으로 어렵고, 개선 여지는 (a) 셀당 다점 평균으로 대표성 노이즈 축소, (b) InSAR 면적평균 관측으로 검증기준 자체 변경에 있다.

### 작은 ALT에서 12cm 오차 = 큰 오차인가 — 맞다, 문헌은 이렇게 처리

ALT 30cm에서 12cm=40%, 16.95cm=56%. 우리 중앙값 48.1cm에서는 12cm=25%, 16.95cm=35%. 문헌 처리법:

1. **심도대 층화 보고**: 전역 단일 RMSE 대신 얕은/중간/깊은 구간별로 나눠 보고(10cm 빈 층화).
2. **상대지표 병기**: cm-RMSE와 함께 MAPE·픽셀 상대불확실성.
3. **저대표성역 마스킹**: 영구동토 부재역·도로 100m내·열카르스트·ice-wedge 극단 제외.
4. **물리적 해석**: 얕은 영역 40% 상대오차의 상당부분은 관측 대표성·정밀도 한계.

우리 조치(신규 실험으로 채택, §7-A): 헤드라인 전역 cm-RMSE 하나만이 아니라 심도 구간별 층화 RMSE + MAPE 병기, 극단 얕은/교란 셀 마스킹 명시.

## 6. 사용자 질문에 대한 데이터 근거 답 (질문 2·3·7)

### 질문 2 — DEM(지형)이 온도·ALT 예측에 의미 있는가

현재 구성에서는 **거의 없거나 오히려 악화**시킨다(셀 ablation 근거):
- 공간블록: M2 지형 단독 skill −11.9%, M1 기후 5.9% → M3 기후+지형 −2.2%(악화).
- LORO: M2 지형 단독 −5.2%, M1 기후 10.8% → M3 기후+지형 8.2%(악화).

이유: (a) 기후 공변량(ERA5 9km)과 지형(30m DEM에서 파생한 6피처)의 **척도 불일치** — 9km 셀 하나에 지형이 이미 평균되어 미세지형 신호가 희석. (b) 알래스카 ABoVE 평탄 툰드라 편중이라 표고·경사 변동이 작고 기후 구배에 압도. 물리적으로 지형(배수·일사·적설재분포)이 ALT에 무관한 것이 아니라, **현재 해상도·파생방식에서 정보를 못 담는 것**이다. 개선 경로: 고해상 유지한 미세지형 파생(습윤지수 TWI·다중스케일 TPI·곡률), 지형×기후 교호항, 그리고 셀이 아닌 고해상 격자 예측. 이는 §7 실험 항목.

### 질문 3 — Stefan 검증자료 + DL 보완으로 더 나은 온도장·ALT 예측 후 참값 비교했나

**부분만.** 우리는 Stefan 방정식의 핵심 항 `sqrt(TDD)`(누적 융해도일의 제곱근)를 **피처로만** 넣었다(era5land_covariates.py:55). Stefan 방정식 자체를 물리 baseline으로 돌려 그 출력을 참값과 비교하거나, Stefan 출력을 DL과 blending한 실험은 **아직 없다.** 문헌 QTP SCE(2025)는 정확히 이 구조(Stefan CatBoost-ET 블렌딩)로 R² 0.873을 얻었다. 우리도 (a) 순수 Stefan baseline 채점, (b) Stefan 잔차를 DL로 학습하는 physics-guided residual을 해볼 가치가 있다. §7 실험 항목으로 채택.

### 질문 7 — 거의 같은 위치인데 ALT가 크게 다른 site 전처리했나

두 종류를 구분해야 한다.
- (a) **정확히 같은 위치(≈11m)의 반복 관측**: `loc_id`(좌표 소수점 4자리 반올림) 단위로 집계 완료. 평균 ALT를 정답, 셀내 SD를 불확실성 라벨로 둠(aggregate_alt_cell.py). 위치당 1행으로 pseudo-replication 제거. loc 내 SD는 평균 3.8cm·중앙값 2.0cm로 작다.
- (b) **다른 위치인데 공변량(9km 기후)이 사실상 같은 경우**: 이건 진짜로 다른 장소라 합칠 수 없다. 같은 공변량 셀에 ALT 34~96cm가 공존하는 이 현상이 **정보병목의 정체**이고, 해결책은 전처리가 아니라 고해상 공변량(InSAR·미세지형)이다.
- **아직 안 한 것**: 통계적 이상치·오측 QC. 현재는 물리적 타당 범위(1~250cm)만 유지하고 within-loc 변동을 자연변동으로 보존한다. "거의 같은 위치에서 ALT가 튀는" 경우가 측정 오류인지 미세지형 진짜 변동인지 판별하는 QC는 향후 과제(다점 셀에서 within-SD 이상치 플래그).

### AOA + conformal UQ 고도화 (P3, Reliability)
- AOA 마스크, DI-RMSE·DI-coverage, 90% 커버리지 표, 불확실성 폭 지도, KPDC 지점 커버리지 점검. LSCP/GeoConformal 국소 conformal은 조사 후보(공개 코드 미확인, 재구현 여유 확보).

### KPDC 독립 검증 (P4, 규칙 충족 + Reliability)
- Council AWS vs ERA5-Land 편향, Council 지중온도 vs 예측 얕은 열구조, Cambridge Bay 대조구 vs 전이 예측, KPDC 지점의 AOA 내외·conformal 구간 포함 여부. KPDC는 **검증·보정 축**으로 전면 배치, 학습셋 무리 편입 금지. 데이터는 로그인+주문요청(승인 대기가 임계 경로) → **오늘 주문**.

### Shallow 3D 열구조 (P5, Primary 차별성)
- GBM 조건장 + depth encoding + CCI 0/1/2/5/10m prior. baseline nearest/IDW/kriging. 검증 필수: leave-one-borehole-out, LORO, 심도별 RMSE, 물리 일관성(depth profile 출렁임·0°C를 ALT로 오해 금지), 심도별 UQ. neural field는 GBM/IDW 이길 때만 본문. ALT(계절 최대융해)와 MAGT(연평균장)·0°C 등온면 개념 분리.

### DL 조건부 확장 (P6, Extension — 게이트)
- Case A (EO/SAR patch): Sentinel-1/2·SAR·DEM 패치 + tabular 융합. 성공조건=공간 CV와 LORO 모두 GBM tabular 개선.
- Case B (T-lite): lagged ALT + 월별 ERA5 시퀀스. 성공조건=temporal holdout에서 persistence·GBM-annual 모두 개선(현재 미통과).
- Case C (3D/4D operator): DeepONet/FNO/physics-guided. 성공조건=leave-one-site/region에서 GBM 조건장 개선 + 보정 UQ.
- foundation embedding: AlphaEarth(GEE 64차원, 즉시), Prithvi/Clay는 비교군. 사전 판정기준(위경도 대조군 skill 14.7% 초과 여부) 고정.

### 사용자 질문에서 파생된 신규 실험
- **A. 심도 구간별 층화 RMSE + MAPE 병기 + 극단 셀 마스킹**(즉시·무비용, §5). 리뷰어 방어 겸 정직성. **최우선.**
- **B. Stefan 물리 baseline 채점 + physics-guided residual DL**(질문 3). QTP SCE 구조 벤치마크.
- **C. 미세지형 재파생 + 지형×기후 교호항**(질문 2). DEM이 정보를 담게 하는 재시도.
- **D. 셀당 다점 평균 대표성 실험**(비가역 하한 규명, §5). FireALT·다점 셀 활용.
- **E. InSAR 면적평균 검증기준 트랙**(검증 프로토콜 전환, §5).
- **F. LORO 지역 확장 재평가**(ALLena·FireALT 러시아분·그린란드로 알래스카 밖 일반화 시험).

## 12. GPT 교차분석 요청

1. **두 트랙 분리(§0)**가 대회 요강("주된 활용")과 상충하지 않는가. 예선을 "스냅샷+로드맵"으로 내는 것이 서류심사에서 감점인가 가점인가.
2. **SOTA 비교 프레이밍(§4)**: "우리는 최소오차가 아니라 누설통제 정직 검증군과 동일 대역"이 데이터 분석 부문에서 설득력 있는가, 더 강한 프레이밍이 있는가.
3. **비가역 하한(§5)** 주장을 심사·논문에서 어느 강도로 내세울지. 하한 규명 실험(§7-D)을 예선에 넣을지 본선으로 미룰지.
4. **미사용 데이터 top5(§8)** 중 예선 스냅샷에 실제 편입할 1~2개 선택. FireALT(화재 축)와 ds2332(독립 벤치마크) 중 우선순위.
5. **DEM·Stefan 재실험(§7-B·C)**의 우선순위. 정보병목 서사를 강화하는가 흐리는가.
6. **위경도 대조군 프레이밍**: "위치 2피처가 물리 공변량 능가"를 "위도=기후 대리"로 과단순화하지 않는 서술.
