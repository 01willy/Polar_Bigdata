# 용어집 (쉬운 설명)

프로젝트에서 쓰는 약자·용어를 비유와 함께 정리.

## 데이터·대상
- **ALT** (Active Layer Thickness, 활성층 두께): 여름에 녹는 표층 깊이(cm). **우리 모델의 주 예측 대상.**
- **MAGT** (Mean Annual Ground Temperature, 연평균 지중온도): 특정 깊이의 1년 평균 온도(°C).
- **borehole / well**: 땅에 뚫어 깊이별 온도를 재는 시추공.
- **CALM** (Circumpolar Active Layer Monitoring): 전 세계 활성층 측정 네트워크. 우리 ALT 데이터 출처.
- **GTN-P**: 전 지구 영구동토 관측망(borehole 온도 + 활성층).
- **공변량 (covariate)**: 예측에 쓰는 "단서" 변수(기온·고도·적설 등). *비유: 몸무게를 키·나이로 추정할 때의 키·나이.*
- **WorldClim bio1/bio4/bio7/bio12**: 무료 전지구 기후격자. bio1=연평균기온, bio4=기온계절성, bio7=연교차, bio12=연강수.

## 모델
- **DL** (Deep Learning, 딥러닝): 신경망 모델.
- **RF** (Random Forest): 수백 개 "의사결정 나무"가 각자 독립 판단 → 평균. *비유: 전문가 100명에게 따로 묻고 평균.*
- **GBM / GBDT** (Gradient Boosting): 작은 나무들을 차례로 쌓되 **앞 나무의 실수를 다음 나무가 보정**. *비유: 앞사람 실수를 뒷사람이 고치는 릴레이.* 표(tabular) 데이터에 강함.
- **IDW** (Inverse Distance Weighting, 역거리 가중): 가까운 측정점일수록 더 믿어 평균. *비유: 내 위치 기온을 가장 가까운 기상대 위주로 추정.* 근처 데이터 없으면 무력.
- **Kriging (크리깅)**: IDW에 "거리에 따라 값이 얼마나 변하는지"(베리오그램)를 반영한 고급판 + 불확실성 제공.
- **PINN** (Physics-Informed Neural Net): 신경망이 **물리식(열전도)** 을 지키도록 강제. (후반 단계)

## 평가
- **CV** (Cross-Validation, 교차검증): 데이터 일부를 **숨기고** 나머지로 학습 → 숨긴 것으로 시험. "외운 게 아니라 진짜 배웠는지" 확인. *비유: 공부한 문제 말고 안 본 문제로 시험.*
  - **무작위 K-fold**: 행을 무작위로 숨김 → 같은 장소의 다른 해가 학습/시험에 섞여 **과대평가(누설)**.
  - **site-disjoint**: 사이트 통째로 숨김.
  - **공간블록(spatial block)**: 지역 덩어리(예 500km)로 숨김 → 근처 사이트 유사성 차단.
  - **LORO** (Leave-One-Region-Out): 국가 하나를 통째 숨김 → "처음 보는 지역 예측" = **전이(transfer)** 측정. 우리의 핵심 지표.
- **누설 (leakage)**: 시험 정답이 학습에 새어들어가 성능이 부풀려지는 것.
- **RMSE / MAE**: 오차 크기. MAE=평균 |오차|. RMSE=큰 실수에 더 벌점(제곱). 단위 cm, **낮을수록 정확**.
- **UQ** (Uncertainty Quantification): "ALT=60cm"가 아니라 "60±15cm"처럼 **확신 정도**까지 제시.
- **변동도/베리오그램 (variogram)**: "두 지점이 멀어질수록 ALT가 얼마나 달라지나" 그래프. 여기서 **range**(서로 무관해지는 거리)로 공간블록 크기를 정함.

## 피처·물리
- **degree-days (도일)**: 하루 기온을 1년 합산. **TDD**(융해, 0°C 위 합) / **FDD**(동결, 0°C 아래 합). *비유: 여름 "총 열량".*
- **Stefan 식**: 활성층 두께 ≈ √(융해 도일 × 토양상수). ALT의 간단한 물리 공식 → TDD가 강력한 예측 단서.
- **log1p / expm1**: ALT가 34~427cm로 치우쳐 있어 `log(ALT)`로 예측 후 되돌림. *비유: 로그 눈금 그래프.*
