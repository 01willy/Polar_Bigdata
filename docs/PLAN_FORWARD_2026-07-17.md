## 3. 새 방향 통합 (Phase B) · 피드백 반영, 둘 다 의미 있음·즉시 가능

### B1. 시계열(timelapse) ALT · 신규 핵심 방향
- **동기(피드백)**: 정적 활동층이 아니라 연도별 ALT 값을 산출해 의미 있는 방법론 제안.
- **왜 의미 있나**: (1) P2에서 검증한 Stefan(ALT=a+E√TDD)의 시변 입력이 곧 그해 TDD다. 물리 모델이 timelapse의 자연스러운 엔진. (2) T-lite에서 시퀀스 DL은 졌지만 **연도별 forcing을 먹인 정적 GBM이 최우수(15.86cm)**. 방법론이 이미 부분 검증됨. (3) 정적 지도를 넘어 추세·변동 탐지라는 동적 산출물 → 대회 차별성.
- **방법**: `alt_era5_temporal.csv`(연도별 e5t_*)를 입력으로, Stefan-물리/GBM으로 **2005-2024 연별 ALT 지도** 생성. 평가는 temporal holdout(뺀 연도 예측).
- **정직한 평가(방법론 기여)**: 대부분 분산은 공간(위치)이다. 따라서 **연내 변동만 분리한 within-site 시계열 skill**(연 anomaly 예측·detrended RMSE)을 별도 보고해 "ALT 변동의 시간 예측 가능분 vs 공간 지배분"을 정량화. 물리모델이 그해 TDD→ALT 반응을 잡는지 검증.
- **산출**: 연별 ALT 지도 애니메이션, ALT 추세(2005-2024 심화 여부), 온난 여름→심융해 귀속, temporal 검증표.
- **주의**: 공간 지배를 숨기지 말 것. 시간 신호는 작지만 실재. 물리 forcing이 외삽 안정성 제공.

### B2. 인터폴레이션(co-kriging) vs 위성-ML 비교 · 교수 요청
- **동기(피드백)**: 위성 기반 방법론과 co-kriging 같은 지구통계 보간을 비교. 교수 권고.
- **왜 의미 있나**: "ML이 좋은 지구통계보다 나은가"는 매핑 논문의 표준 반문. 위성 2차변수를 쓴 co-kriging이 가장 강한 보간 baseline. 결과가 우리 논지를 양방향으로 강화: co-kriging≈GBM이면 정보병목 재확인, GBM/물리>co-kriging(특히 전이)이면 위성-ML·물리 접근 정당화(크리깅은 관측 없는 새 지역 외삽 불가).
- **방법**: 같은 공간블록+LORO CV에서 사다리 비교 · Ordinary Kriging(ALT만) / **Co-Kriging·Kriging with External Drift·Regression Kriging(ALT + 위성 2차변수: InSAR·PolSAR·CCI)** / GBM(공변량) / Stefan 물리. 도구 pykrige·gstools(설치 확인됨).
- **핵심 프레이밍**: 전이(LORO)에서 크리깅은 관측 없는 지역에 못 가지만 물리·ML은 간다 → 위성 융합·물리의 전이 우위 정량화.
- **산출**: 보간 vs 위성-ML vs 물리 사다리표(공간블록·LORO), 방법별 오차 지도.
